機械工程 · 運動學

機械如何
運動？

齒輪轉動、凸輪推動閥門、活塞往復、機器手臂伸展——這些看似複雜的動作，其實都由少數幾種「基本機構」組合而成。機構學就是理解這件事的語言。

230+歷史模型
∞設計可能
機構動態示意 · Kinematics Diagram

機構學研究的是「運動如何傳遞」：旋轉的速度與方向可以改變（齒輪），等速旋轉可以變成任意規劃的往復（凸輪），連桿的長度決定機構的型態。掌握這些原理，你就能看懂並設計任何機械。

Historical Context

機構學是怎麼
發展起來的？

紀元前
中國 · 希臘 · 印度

最早的齒輪

目前已知最早的人造齒輪出土於中國，可追溯到約西元前 4 世紀（戰國時代）。希臘人約在西元前 3 世紀也發展出木製齒輪。西元 1 世紀的安提基特拉機械（Antikythera mechanism）是迄今最複雜的古代齒輪裝置，用於推算天體位置，展示了古代對齒輪傳動的深刻理解。[19]

13–14 世紀
印度 · 中世紀工匠

蝸輪蝸桿的發明

蝸輪蝸桿機構最早在印度出現，用於棉花壓榨機（roller cotton gin）。其關鍵特性——蝸桿可驅動蝸輪，但蝸輪因螺旋角太淺而無法反轉驅動蝸桿（自鎖）——使它成為升降與輸送機械的理想選擇。[21]

1588
Agostino Ramelli · 義大利

《精巧機械圖鑑》

Ramelli 出版圖集，以精美插圖系統記錄水車、起重、幫浦等百餘種機構，是機構知識從「工匠秘技」走向「公開紀錄」的里程碑。[2]

1784
James Watt · 蒸汽時代

瓦特連桿與工業革命

Watt 設計出瓦特連桿（Watt's Linkage）——讓搖臂的弧形運動近似直線，實現雙向作功，蒸汽機效率倍增。這個四連桿設計至今仍出現在汽車後懸吊系統裡。[3]

1837
Robert Willis · 英國劍橋大學

機構學成為獨立學科

Willis 出版《Principles of Mechanisms》，首次以科學分類法整理各類機構，奠定機構學的學科基礎，深刻影響了後來的 Reuleaux。[4]

1853–
Pafnuty Chebyshev · 俄羅斯

車比雪夫連桿

數學家 Chebyshev 設計出連桿點軌跡能高精度近似直線的四連桿，並首次用數學公式分析連桿誤差，開啟了機構「計算合成」的先河。[5]

1875
Franz Reuleaux · 柏林皇家工業學院

機構學的奠基之作

Reuleaux 出版《Theoretische Kinematik》，首次嚴格定義運動副（kinematic pair）、運動鏈（kinematic chain），並親手製作 230 件金屬教學模型，現由 Cornell 大學典藏（KMODDL）。[6,7]

1888
Charles Hoot · 工業化齒輪製造

漸開線齒輪標準化

19 世紀末，漸開線齒形（involute profile）被廣泛採用並逐步標準化。漸開線能確保咬合時的瞬時速比恆定（齒輪嚙合定律），且對中心距偏差不敏感——現代幾乎所有精密齒輪都採用漸開線齒形。[19]

1959
Freudenstein & Sandor

電腦輔助機構設計

率先用電腦解算連桿方程式，從此設計機構不再需要繁瑣的圖解法。這直接促成現代機器人手臂逆運動學（IK）算法的誕生。[8]

今日
機器人 · 仿生機構 · 軟體機器人

老原理，新應用

從工業臂到手術機器人、外骨骼、仿生腿，四連桿、滑塊曲柄、凸輪、齒輪仍是核心設計語彙。MotionGen、OnShape 讓你在瀏覽器裡就能設計並模擬機構。

「機器是由具有確定形狀的固體所組成的組合體，以特定的方式排列，使得機械力可以透過它們來完成特定的工作。」
— Franz Reuleaux, Theoretische Kinematik, 1875

Core Concepts

四個關鍵詞，
搞懂機構學

學機構學之前，先把這四個概念釐清。後面遇到任何機構，都能用這四個詞來分析。

Machine

機械

能輸入能量、透過機構傳遞運動、對外輸出有用工作的系統。[4]

引擎 → 燃料 ▸ 轉動輸出
馬達 + 齒輪 → 扭矩放大
機器臂 → 精確空間定位
Mechanism

機構

機械裡專門負責「傳遞和轉換運動」的部分，由多個剛性構件透過接頭（joints）連接形成運動鏈。機構不在乎力的大小，只在乎「動作怎麼傳遞」。[4,6]

轉動 ↔ 直線 ↔ 擺動 ↔ 間歇
Joint

接頭

兩個構件之間的連接方式，決定兩者能發生什麼樣的相對運動。就像門和門框之間的鉸鏈——只能旋轉，不能滑動。

旋轉接頭 (R)：只能旋轉 → 門鉸鏈、軸承
滑動接頭 (P)：只能平移 → 滑軌、汽缸活塞
學術用語：Kinematic Pair（運動副）。面接觸稱「低對（lower pair）」，點/線接觸稱「高對（higher pair）」，如齒輪齒面、凸輪與從動件。[6]
Degree of Freedom

自由度 DOF

機構裡「獨立可控的運動」有幾個？一個平面剛體本來有 3 個自由度（x、y、旋轉角θ），加入每一個接頭約束，就少一個自由度。[9]

DOF = 1：輸入一個動作，輸出確定 ✓
DOF = 0：完全固定，這是結構 ✗
移動 x移動 y旋轉 θ平面剛體 = 3 DOF
Gear Mechanisms

齒輪：
改變速度與扭矩

齒輪是什麼？

齒輪是透過「齒齒咬合」傳遞力與運動的機構元件，兩個齒輪的齒面屬於高對（higher pair）——接觸是在齒面上移動的線/點接觸。和皮帶、鏈條不同，齒輪是正驅動（positive drive）——不會打滑，傳遞比例精確。

齒輪最重要的功能是：改變轉速、改變扭矩、改變轉軸方向。這三件事是透過齒輪的「齒數比」精確控制的。

速比公式

i = N₂ / N₁ = ω₁ / ω₂

i = 速比（gear ratio）　N₁ = 主動齒輪齒數　N₂ = 從動齒輪齒數
ω₁, ω₂ = 主動、從動角速度。功率守恆（忽略摩擦）：T₁ × ω₁ = T₂ × ω₂

  • 減速增矩：小齒輪驅動大齒輪 → 輸出轉速降低，扭矩增大。例：馬達→機器人輪子。
  • 增速減矩：大齒輪驅動小齒輪 → 輸出轉速提高，扭矩減小。例：腳踏車高速檔。
  • 方向改變：外嚙合兩齒輪旋轉方向相反；內嚙合方向相同；錐齒輪可改變轉軸方向。
  • 複式齒輪列：多組齒輪串聯，可達到非常大的速比。例：鐘錶機構速比可達數千比一。

六種主要齒輪類型

正齒輪
Spur Gear · 平行軸

最常見、最簡單。齒條平行於轉軸，嚙合時有衝擊噪音，不適合高速靜音場合。是學習齒輪的起點。

鐘錶工具機玩具馬達
斜齒輪（螺旋齒輪）
Helical Gear · 平行軸

齒條有傾斜角，齒逐漸咬合，運轉平滑安靜。幾乎所有汽車變速箱都用斜齒輪。缺點是產生軸向推力。

汽車變速箱工業減速機
錐齒輪
Bevel Gear · 相交軸 90°

錐形齒輪，用於傳動方向需要改變的場合，兩軸通常成 90° 相交。

汽車差速器手搖鑽直角傳動
蝸輪蝸桿
Worm Gear · 非相交軸 90°

最大特點：自鎖性——蝸桿能驅動蝸輪，但蝸輪無法反轉驅動蝸桿。速比可達 300:1 以上。

升降機輸送帶機器人關節
行星齒輪組
Planetary / Epicyclic Gear

由太陽齒輪、行星齒輪、環齒輪與行星架組成。固定不同元件得到不同速比，結構緊湊、扭矩密度高。[20]

Cam Mechanisms

凸輪：
把旋轉變成任意運動

凸輪的核心概念

凸輪是一種形狀特殊的旋轉構件。從動件（follower）與凸輪輪廓保持接觸，隨著凸輪旋轉，輪廓的高低起伏「強迫」從動件做出特定運動。

關鍵特點：輸入是等速旋轉，輸出的運動曲線完全由凸輪輪廓形狀決定。設計者可以「程式化」從動件的任意運動歷程——只要把想要的位移曲線轉換成輪廓即可。

凸輪與從動件之間是點或線接觸，屬於高對（higher pair），這和四連桿的面接觸低對不同。[22]

一個旋轉週期內的三種狀態

位移程式（Motion Program）

  • 上升（Rise）：從動件向上移動，凸輪輪廓半徑增大。
  • 停駐（Dwell）：從動件靜止不動，凸輪仍在旋轉，此段輪廓為等半徑圓弧。
  • 下降（Return / Fall）：從動件回到最低位置，輪廓半徑縮小。

三種常見凸輪輪廓

盤型凸輪（Plate Cam）
上升 → 停駐 → 下降 → 停駐

最常見。平面圓盤輪廓，從動件在垂直於旋轉軸的平面內運動。引擎凸輪軸就是最典型的應用。

桶型凸輪（Cylindrical Cam）
沿軸向移動

槽切在圓柱表面，從動件沿平行轉軸方向移動。適合需要大行程直線輸出的場合，常見於紡織機、換刀機構。

偏心凸輪（Eccentric Cam）
簡諧往復運動

把圓形圓盤的中心偏移，從動件做近似簡諧運動。最簡單的凸輪型態，常用於幫浦驅動。

Foundation

四連桿：
最基本的連桿機構

為什麼從四連桿開始？

平面機構裡，三根桿加三個旋轉接頭，DOF = 0，形成完全固定的三角形——這是結構，不是機構。最簡單的可動平面閉迴路機構，就是四連桿。四根桿、四個旋轉接頭，DOF = 1。[9]

決定型態的兩個因素

① 連桿長度比例——決定有沒有桿能做完整旋轉，由 Grashof 定律判斷。

② 固定哪根桿——在滿足 Grashof 條件的前提下，固定不同的桿得到不同型態（運動鏈反轉）。

s + l ≤ p + q

s = 最短桿　l = 最長桿　p, q = 另外兩桿
滿足此條件時，最短桿相鄰的桿（或最短桿本身）可以做完整旋轉。[9,10]

輸入桿與輸出桿

用輸入桿（driver）和輸出桿（follower）描述功能角色更清楚。能連續旋轉的輸入桿叫「曲柄（crank）」，只能擺動的輸出桿叫「搖桿（rocker）」。

應用 01
內燃機

01

開啟工具

前往 linkage.js.org，無需登入。畫面中央會看到預設的四連桿機構，已經在動了。

02

認識介面

黃色節點 = 固定接頭（地桿），灰色節點 = 可動接頭。連桿以直線表示，右側有連桿長度數值輸入欄位。

03

拖拉節點，改變長度

直接拖拉任意節點，連桿長度即時改變，機構運動也跟著變。觀察輸出桿的擺動角度如何隨連桿比例改變。

04

驗證 Grashof 條件

把最短桿調很短（例如長度 = 1），最長桿調很長（例如長度 = 5），觀察哪根桿能做完整旋轉。再反過來調到不滿足 s + l ≤ p + q，觀察機構變化。

05

觀察連桿點軌跡

點擊連桿上任意位置新增「連桿點」，啟動軌跡顯示後，觀察它描繪出什麼曲線（coupler curve）。

06

挑戰任務

嘗試設計一個曲柄搖桿，讓輸出桿的擺動範圍盡量大（接近 180°）。記錄你找到的連桿長度比例。

01

註冊免費帳號

前往 motiongen.io，點擊 Sign Up 建立免費帳號。確認信可能在垃圾信件匣。建議使用 Chrome 或 Edge。

02

建立第一個四連桿

點擊右上角「Add Link」加入連桿，用「Add Joint」在兩根連桿之間建立接頭，連接四根桿形成閉迴路。

03

設定地桿（固定接頭）

點擊一個接頭，選擇「Ground」將它固定。四連桿需要固定兩個接頭（地桿兩端）才能正常運動。

04

加入驅動器（actuator）

點擊輸入桿的旋轉接頭，選擇「Add Rotary Actuator」，設定旋轉速度。按播放，機構開始動。

05

查看連桿點軌跡

點擊任意連桿上的點，選擇「Show Coupler Curve」，看這個點在整個運動週期內描繪的軌跡。

06

進階：位置合成

在畫面上標記連桿要通過的姿態位置，讓軟體自動計算出需要的連桿尺寸——這是 MotionGen 最強大的功能。
